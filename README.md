# Smart Home Light Control

## Overview
This project is a basic Smart Home Light Control system developed using a microcontroller. 
The system allows users to control an LED light using a push button, similar to a simple home light switch.

## Components Used
- ESP32 / Arduino Board
- LED
- Push Button
- Resistor
- Jumper Wires

## Software Used
- Arduino IDE
- Wokwi / Tinkercad (Simulation)

## Working Principle
The push button is connected as an input device and the LED is connected as an output device.
When the button is pressed, the microcontroller detects the input signal and turns ON the LED.
When the button is released, the LED turns OFF.

## Features
- Simple light ON/OFF control
- Microcontroller-based automation
- Easy to understand embedded system project
- Low-cost smart home concept

## Code Explanation
- `ledPin` is configured as an output to control the LED.
- `buttonPin` is configured as an input using `INPUT_PULLUP`.
- The program continuously checks the button state and controls the LED accordingly.

## Applications
- Basic home automation
- Smart lighting systems
- Embedded systems learning projects
